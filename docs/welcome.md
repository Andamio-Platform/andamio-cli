### Install Andamio CLI

1. Download executables from [release page](https://github.com/Andamio-Platform/andamio-cli/releases)

2. Or by building from source (requires Go)

```
git clone https://github.com/Andamio-Platform/andamio-cli
cd andamio-cli
go mod tidy
go build
./andamio-cli
```
